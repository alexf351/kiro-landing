## Why image prompting takes real practice

Modern image models reward specificity. A vague prompt produces a vague image, and a well-constructed prompt produces something close to the picture in your head. Iro turns image prompting into a daily skill: name the subject, define the style, set the framing, choose the lighting, and decide what to vary.

Lessons move from concept (composition, color, lens, mood) into exercises (rewrite, compare, critique). The goal is judgment that transfers across model upgrades, not magic phrases that stop working when a model changes.
## What you practice in Iro

- Prompt structure: subject, style, framing, lighting, lens, mood, era.
- Style transfer techniques that survive model updates.
- Iteration: when to re-prompt, when to inpaint, when to start over.
- Evaluating output for anatomy, text, hands, perspective, and brand fit.
- Picking the right model for the job across Midjourney, DALL·E, SDXL, Imagen, and Flux.

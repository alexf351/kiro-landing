## Why prompt engineering needs active practice

Prompt engineering is not just writing a clever sentence. It is the process of turning an intention into instructions an AI system can follow, then evaluating whether the result is accurate, useful, and safe enough for the task. That requires repetition.

Iro breaks the skill into small decisions: define the goal, provide context, choose constraints, request a useful format, ask for critique, and verify the output. The app reinforces those decisions through exercises, XP, and daily challenges.
## Skills covered

- Prompt structure and specificity.
- Context packing and examples.
- Output formats for briefs, tables, checklists, plans, and drafts.
- Critiquing AI answers and identifying hallucinations.
- Adapting prompts for different tools and use cases.
## Why Iro is different from a prompt library

A prompt library can give you starting points, but it cannot build judgment by itself. Iro is designed to make users practice the thinking behind the prompt: what information matters, what the model may misunderstand, and how to improve the next attempt.
